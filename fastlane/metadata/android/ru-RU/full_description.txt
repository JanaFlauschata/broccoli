Broccoli — это свободное экологичное приложения для создания <b>коллекции рецептов</b>, <b>приготовления блюд без отвлечения внимания</b> и <b>поиска сезонных ингредиентов</b>. Создавайте, собирайте и готовьте!

<b>Легко организовать</b>
• создайте неограниченное количество рецептов
• импортируйте рецепты из ваших любимых блогов
• организуйте рецепты с помощью категорий и хештегов
• доступ к рецептам оффлайн
• резервное копирование рецептов

<b>Готовьте экологично</b>
• узнайте больше о сезонных ингредиентах в вашем регионе с помощью сезонного календаря
• поиск сезонных рецептов в вашей коллекции
• легко определить сезонные ингредиенты

<b>Готовьте без отвлечения внимания</b>
• используйте помощника по приготовлению блюд в полноэкранном режиме во время приготовления блюд
• изменяйте количество ингредиентов

Broccoli является бесплатным для всех и не требует создания аккаунта. Если вам нравится наше приложение для рецептов, вы можете внести пожертвование, чтобы поддержать разработку приложения.

Начните свою коллекцию рецептов прямо сейчас!
